# 用户文档

## 初始数据

### 用户

| 用户名 | 密码 | 角色 | 最长可借天数 | 可借书类别 | 同一时间最大可借数量 |
| -- | -- | -- | -- | -- | -- |
| undergraduate | undergraduate | 本科生 | -1天（用来测试计算逾期款）| 1  | 1 |
| graduate | graduate | 研究生 | 15天 | 1和2 | 2 | 
| admin | admin | 管理员，不能进入借书界面。 | N/A | N/A | N/A |

### 书

| 书名 | 类别 | 可用电子档 | 库存 |
| -- | -- | -- | -- | 
| 1 | 1 | PDF和TXT | 2 |
| 2 | 2 | TXT | 2 |
